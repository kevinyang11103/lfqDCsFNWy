## 前言

欢迎来到本项目的Gitee页面！这是一个基于SpringBoot的微信小程序民宿预约管理系统，作为计算机毕业设计项目，它融合了实用的技术架构和现代化的用户体验。以下是项目的详细介绍。

## 内容介绍

本项目旨在为广大用户提供一个便捷、高效的民宿预约平台。系统通过微信小程序的形式，让用户能够轻松浏览各类民宿信息，实现线上预约与支付。后端采用Java语言和Spring Boot框架，确保系统稳定可靠。此外，我们还关注用户体验，前端使用了JS、Vue和CSS3等技术，让界面美观、操作流畅。

## 技术介绍

- **语言：Java**
- **使用框架：Spring Boot**
- **前端技术：JS、Vue、CSS3**
- **开发工具：IDEA/Eclipse**
- **数据库：MySQL 5.7/8.0**
- **数据库管理工具：phpstudy/Navicat**
- **JDK版本：jdk1.8**
- **Maven: apache-maven 3.8.1-bin**
- **前端环境：Node.Js 12\14\16**
